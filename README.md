### Deskripsi Program 
Program ini merupakan aplikasi berbasis CLI untuk mencatat pengeluaran harian, menghitung statistik sederhana, dan mencatat waktu transaksi secara otomatis. 
 
### Implementasi Kriteria Tugas 
1. **Struktur Kontrol**: Menggunakan `while loop` untuk menu utama, `if-elif-else` untuk logika navigasi, dan `try-except` untuk validasi input. 
2. **Struktur Data**: Menggunakan `List` untuk menampung sekumpulan data, dan `Dictionary` untuk menyimpan detail setiap transaksi. 
3. **Library**: Menggunakan library `datetime` (waktu transaksi) dan `statistics` (menghitung rata-rata). 
 
### Cara Menjalankan 
1. Pastikan Python terinstall di laptop. 
2. Jalankan perintah: `python expense_tracker.py`
